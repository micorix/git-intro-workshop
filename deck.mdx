# Commit & push

```shell script
git commit -m "My commit message"
```
``` shell script
git push
```

<Steps>
<div>

## When pushing for the first time
```shell script
git push -u origin my_branch
```
* where `my_branch` is usually `master` or `main`.

</Steps>
